## Introduction

    前端vueCli + vue2 + iView 整合项目

## Features

- Login / Logout
- Permission Authentication
    - A list of filters
    - Permission to switch
- i18n
- Components
    - Rich Text Editor
    - Markdown Editor
    - City Cascader
    - Photos preview and edit
    - Draggable list

## Getting started
```bush
    npm install

    npm run dev
```
